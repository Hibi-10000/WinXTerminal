WinXTerminal [ver1.0]

Win+Xメニューのコマンド プロンプトやWindows PowerShellをWindows ターミナルに置き換えます
※Windows10向けです
  Windows11のWinXメニューは標準でWindows ターミナルになっています
  Windows8/8.1で動作するかは不明です
  Windows7にはWindows ターミナルをインストールできません

動作確認
Windows10 Pro 21H2 x86_64
